# 📋 Менеджер задач (Flutter)

Минималистичное мобильное приложение для управления задачами с поддержкой приоритетов, дедлайнов и тёмной темы.

![Dart](https://img.shields.io/badge/Dart-3.x-blue?logo=dart)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Возможности

| Функция | Описание |
|---|---|
| ➕ Добавление задач | Название, описание, дедлайн, приоритет |
| ✏️ Редактирование | Изменение любого поля уже созданной задачи |
| ✅ Чекбоксы | Отметить задачу выполненной — с мотивирующим уведомлением |
| 🗑️ Удаление | Удаление задачи из списка |
| 🔥 Приоритеты | `MAX` → `MED` → `MIN` с цветовой индикацией |
| ⚠️ Просроченные | Визуальная подсветка и сортировка вверх |
| 📅 Дедлайн | Выбор даты и времени, отображение в списке |
| 🔍 Фильтрация | «Все» / «Активные» / «Выполненные» |
| 📊 Статистика | Счётчики, прогресс-бар, количество просроченных |
| 🌙 Тёмная тема | Переключение в один клик, сохранение настройки |
| 💾 Персистентность | Данные сохраняются через `shared_preferences` |

## 🚦 Логика сортировки

Задачи всегда отображаются в следующем порядке:

```
1. ⚠️  Активные просроченные   — по дедлайну (ближайший вверх)
2. 🔥  MAX-приоритет            — по дедлайну
3. 🟠  MED-приоритет            — по дедлайну
4. 🟢  MIN-приоритет            — по дедлайну
5. ✅  Выполненные              — по дедлайну
```

> Задачи без дедлайна внутри группы располагаются в конце.

---

## ⚠️ Логика просроченных задач

Задача считается **просроченной** если:
- ❌ Не выполнена
- 📅 Имеет дедлайн
- ⏰ Дедлайн уже прошёл

Что происходит с просроченными задачами:
- Плитка выделяется красной рамкой и фоном
- В подзаголовке появляется иконка `alarm_off` и красный текст `⚠ дата`
- В детальном просмотре — предупреждающий баннер
- В шапке экрана — отдельный счётчик «Просрочено»
- При сортировке выходят **наверх** (приоритетнее незавершённых задач в срок)

После выполнения — статус просроченности снимается автоматически.

## 🛠️ Установка и запуск

### Требования

- [Flutter SDK](https://flutter.dev/docs/get-started/install) `≥ 3.0`
- Dart `≥ 3.0`
- Android Studio / VS Code с плагином Flutter

### Зависимости (`pubspec.yaml`)

```yaml
dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.2.0
  flutter_localizations:
    sdk: flutter
```

### Запуск

```bash
# Клонирование
git clone https://github.com/Pavel1778/task_manager_flutter_ru.git
cd task_manager_flutter_ru

# Установка зависимостей
flutter pub get

# Запуск на устройстве/эмуляторе
flutter run

# Сборка APK
flutter build apk --release
```
