# Менеджер задач

Мобильное приложение для управления задачами, разработанное на Flutter.

## Возможности

- ✅ Создание, редактирование и удаление задач
- ✅ Установка сроков выполнения с датой и временем
- ✅ Три уровня приоритета (MAX, MED, MIN)
- ✅ Фильтрация задач (Все, Активные, Завершённые)
- ✅ Тёмная и светлая тема оформления
- ✅ Автоматическое сохранение данных
- ✅ Отметка просроченных задач

## Установка и запуск

1. Убедитесь, что у вас установлен Flutter SDK
2. Клонируйте репозиторий
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   ```bash
   flutter run
   ```

## Структура проекта

```
lib/
├── main.dart                    # Точка входа приложения
├── models/
│   └── task.dart               # Модель данных Task
├── services/
│   ├── task_manager.dart       # Управление задачами
│   └── theme_provider.dart     # Управление темой
├── screens/
│   └── task_screen.dart        # Главный экран
└── widgets/
    ├── task_tile.dart          # Виджет карточки задачи
    └── add_task_dialog.dart    # Диалог добавления задачи
```

## Технологии

- **Flutter** - фреймворк для создания кросс-платформенных приложений
- **Provider** - управление состоянием приложения
- **SharedPreferences** - локальное хранение данных
- **Material Design 3** - современный дизайн
